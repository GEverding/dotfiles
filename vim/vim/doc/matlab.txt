                  <div class="highlight"><pre><div class='line' id='LC1'>MatlabEditionFiles			 	*matlab* *MatlabEditionFiles*</div><div class='line' id='LC2'>A set of files useful to edit Matlab files.</div><div class='line' id='LC3'><br/></div><div class='line' id='LC4'>Included is : </div><div class='line' id='LC5'>1. Syntax highlighting 					|matlab-syntax| </div><div class='line' id='LC6'>2. Using the matchit.vim script 			|matlab-matchit|</div><div class='line' id='LC7'>3. Correct indentation 					|matlab-indent| </div><div class='line' id='LC8'>4. Integration of the mlint Matlab code checker 	|matlab-compiler|</div><div class='line' id='LC9'>5. Tag support 						|matlab-tags|</div><div class='line' id='LC10'>6. Installation details 				|matlab-instal|</div><div class='line' id='LC11'><br/></div><div class='line' id='LC12'>================================================================================</div><div class='line' id='LC13'>1. Syntax highlighting						*matlab-syntax*</div><div class='line' id='LC14'><br/></div><div class='line' id='LC15'>syntax/matlab.vim : Updates the matlab.vim syntax file provided in the vim</div><div class='line' id='LC16'>distribution :</div><div class='line' id='LC17'>- highlights keywords dealing with exceptions : try / catch / rethrow</div><div class='line' id='LC18'>- highlights keywords dealing with class definitions : classdef / properties /</div><div class='line' id='LC19'>&nbsp;&nbsp;methods / events</div><div class='line' id='LC20'><br/></div><div class='line' id='LC21'>================================================================================</div><div class='line' id='LC22'>2. Correct settings in order to use the matchit.vim script	*matlab-matchit*</div><div class='line' id='LC23'><br/></div><div class='line' id='LC24'>The matchit.vim extends the % matching and enables to jump through matching</div><div class='line' id='LC25'>groups such as &quot;if/end&quot; or &quot;switch/end&quot; blocks (see :help matchit in vim)</div><div class='line' id='LC26'><br/></div><div class='line' id='LC27'>ftplugin/matlab.m provides the suitable definition for b:match_words in order</div><div class='line' id='LC28'>to jump between if/end, classdef/end, methods/end, events/end, properties/end,</div><div class='line' id='LC29'>while/end, for/end, switch/end, try/end, function/end blocks</div><div class='line' id='LC30'><br/></div><div class='line' id='LC31'>================================================================================</div><div class='line' id='LC32'>3. Correct indentation 						*matlab-indent*</div><div class='line' id='LC33'><br/></div><div class='line' id='LC34'>&nbsp;indent/matlab.vim : Updates the matlab.vim indention file</div><div class='line' id='LC35'>provided in the vim distribution.  This script provides a correct indentation</div><div class='line' id='LC36'>for :</div><div class='line' id='LC37'>- switch / end, try / catch blocks</div><div class='line' id='LC38'>- classdef / methods / properties / events</div><div class='line' id='LC39'>- mutli-line (lines with line continuation operator (...))</div><div class='line' id='LC40'><br/></div><div class='line' id='LC41'>This script has been tested with the Matlab R2008a release on many files and</div><div class='line' id='LC42'>the result of indentation compared to the one provided by the Matlab Editor</div><div class='line' id='LC43'>(using the &#39;indent all functions&#39; option).</div><div class='line' id='LC44'><br/></div><div class='line' id='LC45'>NOTE : to work correctly, this script need the matchit.vim (vimscript#39) to</div><div class='line' id='LC46'>be installed. (see also |matchit|)</div><div class='line' id='LC47'><br/></div><div class='line' id='LC48'>================================================================================</div><div class='line' id='LC49'>4. Integration of the mlint Matlab code checker		 	*matlab-compiler*</div><div class='line' id='LC50'><br/></div><div class='line' id='LC51'>compiler/mlint.m provides the settings to use mlint (Matlab code ckecker) and</div><div class='line' id='LC52'>puts the messages reported in the quickfix buffer.</div><div class='line' id='LC53'><br/></div><div class='line' id='LC54'>Whenever you want to check your code, just type :make and then :copen and vim</div><div class='line' id='LC55'>opens a quickfix buffer which enables to jump to errors (using :cn, :cp or</div><div class='line' id='LC56'>Enter to jump to the error under the cursor : see |quickfix| in vim)</div><div class='line' id='LC57'><br/></div><div class='line' id='LC58'>================================================================================</div><div class='line' id='LC59'>5. Tag support							*matlab-tags* </div><div class='line' id='LC60'>The .ctags file (in the matlab.zip) defines the Matlab language so that the</div><div class='line' id='LC61'>exuberant ctags (http://ctags.sourceforge.net ) can construct the tag file :</div><div class='line' id='LC62'>you can now jump to tags (using CTRL-] (or CTRL-$ if using Windows) and go</div><div class='line' id='LC63'>back again (CTRL-T) See |tags| in vim.</div><div class='line' id='LC64'><br/></div><div class='line' id='LC65'>These scripts have been tested using gvim 7.2 and Matlab R2008a on Windows.</div><div class='line' id='LC66'><br/></div><div class='line' id='LC67'>================================================================================</div><div class='line' id='LC68'>6. $HOME/vimfiles/doc directory.</div><div class='line' id='LC75'>&nbsp;&nbsp;And then run the command to create help for matlab : &gt;</div><div class='line' id='LC76'>	:helptags $HOME/vimfiles/doc</div><div class='line' id='LC77'>&lt; Then the command &gt;</div><div class='line' id='LC78'>	:help matlab</div><div class='line' id='LC79'>&lt; is available</div><div class='line' id='LC80'><br/></div><div class='line' id='LC81'>- Syntax highlighting </div><div class='line' id='LC82'>&nbsp;&nbsp;Copy syntax/matlab.vim to your $HOME/vimfiles/syntax</div><div class='line' id='LC83'>&nbsp;&nbsp;directory</div><div class='line' id='LC84'><br/></div><div class='line' id='LC85'>- Correct settings in order to use the matchit.vim script</div><div class='line' id='LC86'>&nbsp;&nbsp;In your vimrc file, add the following line : &gt;</div><div class='line' id='LC87'>	source $VIMRUNTIME/macros/matchit.vim</div><div class='line' id='LC88'>&lt;</div><div class='line' id='LC89'>&nbsp;&nbsp;And copy ftplugin/matlab.vim to your $HOME/vimfiles/ftplugin directory</div><div class='line' id='LC90'><br/></div><div class='line' id='LC91'>- Correct indentation </div><div class='line' id='LC92'>&nbsp;&nbsp;In your vimrc file, add the following lines : &gt;</div><div class='line' id='LC93'>	source $VIMRUNTIME/macros/matchit.vim</div><div class='line' id='LC94'>&nbsp;&nbsp;&nbsp;&nbsp;filetype indent on</div><div class='line' id='LC95'>&lt;</div><div class='line' id='LC96'>&nbsp;&nbsp;And copy indent/matlab.vim to your $HOME/vimfiles/indent directory</div><div class='line' id='LC97'><br/></div><div class='line' id='LC98'>- Integration of the mlint Matlab code checker with the :make command</div><div class='line' id='LC99'><br/></div><div id='LC102'>&lt;</div><div class='line' id='LC103'>&nbsp;&nbsp;And copy compiler/mlint.vim to your $HOME/vimfiles/compiler directory</div><div class='line' id='LC104'><br/></div><div class='line' id='LC105'>- Tag support</div><div class='line' id='LC106'><br/></div><div class='line' id='LC107'>&nbsp;&nbsp;Copy the .ctags to your $HOME directory.
